蓝桥杯 & 洛谷 & LeetCode 算法刷题仓库，记录算法学习与备赛全过程

---

## 🎯 仓库简介
本仓库用于系统整理**蓝桥杯备赛、洛谷刷题、LeetCode 每日一题**的代码与思路，按算法分类归档，方便复习、复盘与查阅。

---

## 📂 目录结构
├── 01-basic/ # 基础算法（排序、二分、前缀和、双指针等）
├── 02-search/ # 搜索算法（DFS、BFS、回溯、剪枝等）
├── 03-dp/ # 动态规划（背包、线性 DP、区间 DP 等）
├── 04-greedy/ # 贪心算法
├── 05-math/ # 数学与数论（GCD、快速幂、组合数等）
├── 06-string/ # 字符串处理（KMP、哈希、字典树等）
├── 07-structure/ # 数据结构（栈、队列、并查集、线段树等）
├── 08-graph/ # 图论（最短路、最小生成树、拓扑排序等）
├── 09-simulate/ # 模拟与暴力
├── 10-practice/ # 真题与套题（蓝桥杯省赛 / 国赛、洛谷试炼场等）
└── README.md # 仓库说明
